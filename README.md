# Delphi Calculator

A quick calculator project I threw together to learn more about Delphi.

## Supported Operators
- \+
- \-
- \*
- /
- ^
- %
- mod (same as %)

## Supported Functions

- min(arg1, arg2)
- max(arg1, arg2)
- round(min, max)
- abs(arg)
- sqrt(arg)
- round(arg)
- remainder(arg)

## Supported Constants
- pi
- e
